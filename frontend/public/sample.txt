3. Enter valid password.

Bug Report Example
--------------------------------------------------
Bug ID: BUG_102
Title: Login button is unresponsive on mobile.
Severity: High

Description:
The login button does not register a click event on smaller screens (below 420px).

Steps To Reproduce:
1. Open website on mobile.
2. Try clicking "Login".
3. Nothing happens.

Expected:
Login process should start normally.
